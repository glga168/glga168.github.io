Going over Javascript

Why is Javascript regarded as an "Easy Language?"

There are many reasons Javascript is regarded as such an easy programming language to learn. First, it is widely standardized across the web and has a very flexible code, meaning newbies can pick it up easily. It is also very easy to edit if an error occurs, and has lots of available tools to help the programmer. Lastly, it is a very fast and powerful language.
