Programming Language for Dummies

Programming language is a formal language used to communicate instructions to a mahcine, in most common usages, a computer. It can also be used to make programs or express a machine's behavior. Types of Programming Language

What types of programming language are there?

There are many programming languages out there, but here are ten well-known languages. Ruby, Python, SQL, Java, C++, Javascript, PHP, Objective C, C#, and C Language.

What are these programming languages used for?

Ruby and Python are both scripting languages used for developing websites and mobile apps. PHP is a scripting language used for app development, and is used to power Facebook. C Language, the oldest and most widely used programming language, was used to provide the essentials for many of the languages shown here, such as C++ and Python. It is used in many computer internal systems. Objective C is a general-purpose language used by Apple to power its OS X and IOS, while C# is used by Microsoft to power its programs. Java is another very popular language that is commonly used for web-based content, computer games, and apps. SQL is a special-purpose language used for looking through database information. Lastly, there is Javascript, one of the easiest programming languages to learn, that is used for developing games and desktop applications.
